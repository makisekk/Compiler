# Compiler
SysY(C语言子集)到LLVM-IR中间表示的编译器。  
A SysY-to-LLVM-IR compiler.

## LLVM-IR code version
+ LLVM 10

## Java environment
+ JDK 1.8

## Input file
+ /src/testfile.txt

## Output file
+ /src/llvm_ir.txt

## Generate LLVM-IR code
+ run Compiler.main()
